# Application-Gestion-pharmacie
J'ai développé une application de bureau en utilisant C# pour mettre en œuvre mon projet et SQL Server Management Studio (SSMS) pour la création de la base de données. Cette application répond aux besoins quotidiens des pharmaciens en leur offrant des fonctionnalités essentielles telles que l'authentification, la création, la lecture, la mise à jour et la suppression des données (CRUD), l'impression des factures et le paiement en ligne par carte de crédit.
Pour la manipulation des données, j'ai combiné la programmation orientée objet et le T-SQL. J'ai également conçu toutes les interfaces réactives en utilisant uniquement les outils Windows Forms. Grâce à cela, les pharmaciens peuvent facilement naviguer dans l'application et effectuer toutes les tâches dont ils ont besoin de manière efficace et rapide.
